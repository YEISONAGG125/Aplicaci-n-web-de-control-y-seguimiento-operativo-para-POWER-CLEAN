# Aplicaci-n-web-de-control-y-seguimiento-operativo-para-POWER-CLEAN
Descripción del proyecto:
El proyecto “Aplicación Web de Control y Seguimiento Operativo para la empresa Power Clean" es un citio web permite gestionar, coordinar y supervisar los procesos operativos y administrativos en un solo sistema centralizado.

Cómo ejecutarlo:
Tener un navegador para descargar la aplicacion y poder aceder al citio web
